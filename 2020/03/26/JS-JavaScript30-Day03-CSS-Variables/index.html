第三天的挑戰要來認識－CSS變數
上完這課對我來說意外的長知識( 也可能是我孤陋寡聞 QQ )
之前只知道用 SASS 時可以使用變數，沒想到現在純 CSS 也行！

目標

  1. 了解 CSS 變數
  2. 利用 JS 操作 CSS 變數，來改變樣式

筆記

CSS 變數宣告

在任何選擇器中
使用開頭「- -」，後接變數名稱（英文或數字）跟值
如需全域宣告，可以用 :root（＜html＞ 的偽類） 來宣告全域變數

呼叫 CSS 變數

使用「 var(變數名稱) 」在選擇器中來呼叫變數

也可以賦予預設值(選填)，避免變數未宣告

JS 修改變數值

更新的值要記得加上單位!
