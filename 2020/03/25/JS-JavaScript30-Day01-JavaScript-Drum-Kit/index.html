【JS】 JavaScript30－Day01 JavaScript Drum Kit

發表於 2020-03-25 更新於 2020-03-26 分類於 JavaScript30

作品 Demo
開始 JS 30 第一天！
第一天課程是要做出～藉由按下鍵盤按鍵來觸發對應音效跟動畫

目標

  1. 鍵盤事件觸發音效播放
  2. 樣式改變
  3. 監聽動畫事件結束，樣式回復初始樣式

筆記

鍵盤監聽事件－keydown

使用者按下按鍵時觸發的事件
這邊會運用 keyCode ，就是按鍵所對應的數字
像是 A → 65 、S→ 83 等等 ( 可參考影片介紹的 keycode.info )

data-*

data-* 是 Html 5 的新特性可以讓使用者自定義屬性
使用過後發現真是個好東西！
用來在 Html 標籤中塞資料，不論用 JS 或是 CSS 抓取都可以使用
( 寫了個範例給大家參考 ➡️ 程式範例

綜合上面兩者
可以利用 keydown 事件取得 keyCode
由於 keyCode 都已經被寫入 data-key 中了
所以在 JS 就可以帶入 keyCode 值到 data-key 去抓取元素

audio 控制

音樂播放就交給 audio 處理
設定指定音效播放外，還要讓音效都從起始點播放

  1. audio.play() ： 播放
  2. audio.currentTime＝seconds 設定從哪一秒開始撥放

element.classList

接下來就是當鍵盤被觸發，要改變元素的樣式
這部分會使用 element.classList.add(class) 加入樣式　　

先取得所有 class 等於 key 的元素
在對每個元素加上監聽 transition 的事件

transition 事件結束判斷可以印 console.log 查看
如下圖所示
就會發現 TransitionEvent 結束在 propertyName 為 “transform” 的事件
所以就在結束的時間點移除樣式
